# Images Directory

このディレクトリには、アプリで使用する画像ファイルを配置してください。

Place your app's image files in this directory.

## 推奨される画像形式 / Recommended Image Formats

- PNG
- JPG/JPEG
- SVG
- WebP

## ファイル命名規則 / File Naming Conventions

わかりやすいファイル名を使用してください：

Use descriptive file names:

- `logo.png`
- `hero-image.jpg`
- `icon-menu.svg`
- `screenshot-1.png`
